# 多功能AI聊天助手

一个功能丰富、界面美观的AI聊天助手，支持多种AI平台集成，具有网络安全管理、记忆模块、任务管理等功能。

## 🌟 功能特点

### 核心功能
- **多平台支持**：兼容多种AI平台API，如心流AI、OpenAI等
- **美观界面**：支持多种主题切换（浅色、深色、蓝色）
- **流式输出**：支持AI响应的流式显示，提升交互体验
- **对话管理**：支持对话历史的保存、导入和导出

### 网络安全
- **域名白名单/黑名单**：可自定义允许访问的域名
- **IP白名单/黑名单**：可自定义允许访问的IP地址
- **SSL验证**：可配置是否启用SSL证书验证
- **安全日志**：记录所有网络请求的安全状态

### 记忆模块
- **个人信息管理**：可保存和管理个人信息
- **任务记录管理**：支持添加、编辑、删除任务
- **记忆导出/导入**：支持记忆数据的备份和恢复
- **对话记忆**：自动保存对话历史到记忆文件夹

### 网络监控
- **网络状态检测**：实时监控网络连接状态
- **IP地址显示**：显示本地IP地址
- **Ping延迟检测**：测试网络延迟
- **上传下载速度监控**：实时显示网络速度

### 调试功能
- **详细日志**：记录应用程序运行状态
- **调试模式**：可开启/关闭调试信息
- **API状态跟踪**：记录所有API请求的状态和响应时间
- **日志导出**：支持日志的复制和清空

## 📋 系统要求

- Python 3.6+
- Windows、macOS、Linux

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python universal_chatbot_beautiful_with_platform_manager.py
```

## 🛠️ 配置说明

### 配置文件

程序启动后会在软件所在目录生成`chatbot_config.ini`配置文件，包含以下主要配置项：

```ini
# 外观设置
主题 = light
字体大小 = medium

# 联网搜索设置
启用联网搜索 = True
搜索引擎 = Bing
结果数量 = 5

# 安全设置
启用SSL证书验证 = True
域名白名单 = api.openai.com,apis.iflow.cn,api.deepseek.com,baidu.com,bing.com,duckduckgo.com

# 对话设置
最大历史消息数 = 2000
```

### 平台配置

首次运行时，程序会使用默认配置（心流AI）。你可以通过界面上的"添加平台"和"编辑平台"按钮来管理AI平台配置。

每个平台配置包含：
- **平台名称**：自定义平台名称
- **API密钥**：平台的API密钥
- **API地址**：平台的API Base URL
- **模型列表**：平台支持的模型名称
- **启用状态**：是否启用该平台

## 📱 界面说明

### 左侧面板
- **配置设置**：选择AI平台、API地址、API密钥、模型
- **调试控制**：启用/关闭调试模式、自动滚动、自动保存等
- **调试信息**：显示程序运行日志和网络状态

### 右侧面板
- **对话标签**：显示与AI的对话内容
- **搜索结果标签**：显示联网搜索结果
- **输入区域**：输入聊天内容

### 菜单栏
- **文件**：导出/导入对话历史、记忆管理
- **设置**：综合设置对话框
- **帮助**：关于和使用帮助

## 🔧 高级功能

### 自定义主题

程序支持三种主题：
- `light`：浅色主题
- `dark`：深色主题
- `blue`：蓝色主题

可通过设置菜单或配置文件修改主题。

### 网络搜索

程序支持自动检测需要搜索的关键词，并使用配置的搜索引擎进行搜索。可配置：
- 搜索触发关键词
- 搜索灵敏度
- 搜索结果数量
- 搜索引擎选择

### 任务管理

可通过记忆菜单打开任务管理器，支持：
- 添加新任务
- 编辑任务详情
- 设置任务状态和优先级
- 搜索和过滤任务
